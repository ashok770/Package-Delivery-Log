API Access Documentation

These endpoints are designed for system integration (e.g., automated inventory management).

Always use an API tool (like Postman or cURL) for these endpoints. Accessing them directly in the browser will show raw JSON.
GET /api/users
Retrieve All Users

Lists all Student and Staff users in the system.

Output:

[{"userId": 1, "name": "Admin Mailroom", ...}]
POST /api/packages
Log New Package

Creates a new package entry. Requires **staffId** in query parameters.

Input Body (JSON):

{
"sender": {"userId": 1},
"receiver": {"userId": 12},
"description": "New Package"
}
